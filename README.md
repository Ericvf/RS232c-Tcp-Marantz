# Dotnet-RS232c-lan
Control computer monitors and TV's using RS232 commands over LAN. 

## Commands
- PowerControl
- InputModeSelection
- Brightness
- Size
- ColorMode
- WhiteBalance
- RContrast
- GContrast
- BContrast
- TelnetUsername
- TelnetPassword
- ThermalSensorSetting
- Model
- SerialNo
- Auto
- Reset
- Volume
- Mute
- TemperatureSensor
- Temperature
